# AgendaMed

Plateforme de gestion d'agenda médical comprenant un backend Spring Boot et un frontend Angular.

## Structure du Projet

- **backend/** : Service backend développé avec Spring Boot (Java).
- **frontend/** : Interface utilisateur développée avec Angular.

## Prérequis

- Java 17+
- Node.js & npm
- Maven (pour le backend)
- Angular CLI (pour le frontend)

## Installation et Lancement

### Backend
1. Naviguer vers le dossier `backend/agenda-service`.
2. Exécuter `mvn clean install`.
3. Lancer l'application avec `mvn spring-boot:run`.

### Frontend
1. Naviguer vers le dossier `frontend`.
2. Exécuter `npm install`.
3. Lancer le serveur de développement avec `npm start`.
